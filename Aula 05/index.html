HTML

O HTML é a estrutura ou o "esqueleto" de uma página web. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação. É através dele que você define o que é um título, um parágrafo, uma imagem ou um link.

CSS

O CSS é responsável pela estética e pelo design. Sem ele, todos os sites pareceriam documentos de texto simples dos anos 90. Ele permite controlar cores, fontes, espaçamentos e o posicionamento dos elementos criados no HTML.

JavaScript

O JavaScript é a lógica e o comportamento. É uma linguagem de programação real que permite criar elementos dinâmicos e interativos. Se algo se move, muda de cor ao clicar ou carrega novos dados sem atualizar a página, o JS provavelmente está envolvido.
