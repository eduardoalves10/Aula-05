Linguagens de programação

Uma linguagem de programação é um conjunto de regras (sintaxe) e símbolos que permitem a um programador escrever código fonte. Esse código é então traduzido (por compiladores ou interpretadores) para o código de máquina (binário), que é a única coisa que o processador entende.
